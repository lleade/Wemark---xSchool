# xSchool Landing Page

A pixel-perfect and fully responsive landing page built with Next.js based on the provided Figma design.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Server-Side Data Fetching

## Features

- Responsive design
- Pixel-perfect implementation
- API-driven content
- Functional About xSchool slider
- Functional Marquee (text slider)
- Functional FAQ accordion

## API

https://admin.wemark.pro/api/tasks-xschool/

## Getting Started

Install dependencies:

```bash
npm install
````

Run the development server:

```bash
npm run dev
```

Open http://localhost:3000
